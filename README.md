# Tic Tac Toe: Grid Battle Arena

A modern Tic Tac Toe game built with React and Vite, featuring an AI-powered bot opponent, responsive gameplay, and a clean user interface. Challenge the bot, test your strategy, and compete for victory in a fast-paced grid battle experience.

## Features

- Single-player mode against an AI bot
- Classic Tic Tac Toe gameplay
- Fast and lightweight React + Vite architecture
- Responsive design for desktop and mobile devices
- Real-time win, loss, and draw detection
- Quick restart functionality
- Modern and intuitive user interface

## Gameplay

Players take turns placing their marks on a 3×3 grid. The objective is to align three identical marks horizontally, vertically, or diagonally before the opponent does. The integrated AI bot analyzes the board and responds automatically, creating a challenging and engaging single-player experience.
